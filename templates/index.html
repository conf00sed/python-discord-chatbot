Welcome to Fuse Bot

A multipurpose discord bot by @Fuse#0420

Moderation

All the necessarry tools for moderating any servers from small to large. This module includes all standard and required commands for moderating such as kick, mute, ban, purge and much more with customisable settings!

Economy

The ecocnomy module allows users to have a perserver economy system with different ways to spend and use the money they earn. This also directly works with the giveaways module

Leveling System

A comprehensive leveling system thats different every server the bot is in! This level module will allow you to give or take experience from users, setup automatic level roles and compare ranks on the server leaderboard

Fun & Giveaways

These modules will allow for more fun and activity in the server through the use of fun commands and giveaways. This includes a very easy giveaway creator and deeply integrated games that provide per-server economy
